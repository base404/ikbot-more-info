# 爱看机器人影视简介与评分增强 (Tampermonkey Script)

这是一个为 **爱看机器人 (ikanbot.com)** 播放页面量身打造的影视简介与评分增强篡改猴 (Tampermonkey) 脚本。

通过本脚本，您可以在每个影视播放页的详情区域，直接获取来自权威书影音平台的评分、评价人数以及详细的剧情简介。

---

## 🌟 核心特色

1. **🚀 默认极速首选 NeoDB (源: NeoDB)**
   * 默认采用联邦宇宙开源影视数据库 **NeoDB (neodb.social)** 接口。
   * **免人机验证码**，国内直连响应速度极快（毫秒级），影视条目覆盖率极庞大（包括国内外极其小众或最新的影视）。

2. **🛡️ 三源强力联合兜底机制**
   * 首次打开页面时，脚本会自动按顺序启动三源兜底：**NeoDB API $\rightarrow$ 98dou API $\rightarrow$ 豆瓣原生网页**。
   * 无论哪个源出现问题，脚本都会在后台静默尝试下一个源，直至成功把数据呈现在您面前。

3. **⇄ 智能 [手动控源] 与 [一键换源]**
   * 页面渲染后，在右上角常驻 **`换源 ⇄`** 按钮。
   * 如果当前源的数据有缺失（例如电影在当前平台暂无评分或简介），您可以点击 `换源 ⇄`。此时脚本将进入**手动控源模式**，直接精准请求下一个指定的源（不再自动静默降级），以便您自主获取最想要的结果。

4. **🔗 永久常驻的直达详情链接**
   * 不论加载成功、加载中还是加载失败，右上角都永久保留直达详情页的按钮（如 **`直达豆瓣 ↗`** 或 **`直达NeoDB ↗`**），并且会根据您当前所选的源动态变化。
   * 当豆瓣由于安全策略拦截（网页被重定向到验证码页面）时，脚本会 100% 精准识别报错，用户只需点击右上角 `直达豆瓣 ↗` 去完成一次滑块验证，点击 `重试 ↻` 即可瞬间恢复抓取！

5. **⚡ 智能缓存与展开收起**
   * **缓存机制**：成功获取的数据会在本地缓存 7 天；如果是“暂无简介”等空数据，仅缓存 1 天以方便快速同步最新的数据库。
   * **折叠体验**：简介过长时会自动进行限宽折叠，并提供 `[展开]/[收起]` 按钮，不破坏原有页面布局。

## 📥 安装步骤

1. 在浏览器中安装 [Tampermonkey (篡改猴)](https://www.tampermonkey.net/) 插件。
2. 打开最新的脚本代码。
3. 点击浏览器右上角的 Tampermonkey 图标，选择 **“添加新脚本”**。
4. 将编辑器中的默认代码替换为脚本中的全部代码。
5. 保存并启用即可！

## 🛠️ 技术路线与防爬策略

* **98dou API**：国内影视采集公益 API，单 IP 限制每小时 30 次请求。
* **豆瓣网页抓取**：通过 `GM_xmlhttpRequest` 跨域解析豆瓣原生 HTML 主页。
  * *注：脚本头部特别配置了 `// @connect sec.douban.com`，可以完美拦截并识别豆瓣的伪 200 OK 验证码重定向页面，并在界面显示友好的滑块验证提示。*
* **NeoDB API**：无限制的开放 API，接口为干净的 JSON 元数据，不仅提供原始评分和简介，还提供关联的豆瓣详情页资源用于完美转换。
