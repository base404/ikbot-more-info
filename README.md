# 爱看机器人影视简介与评分增强 (Tampermonkey Script)

GitHub 仓库: [base404/ikbot-more-info](https://github.com/base404/ikbot-more-info)

这是一个为 **爱看机器人 (ikanbot.com)** 播放页面量身打造的影视简介、类型与评分增强篡改猴 (Tampermonkey) 脚本。

通过本脚本，您可以在每个影视播放页的详情区域，直接获取来自开源与第三方开放平台的评分、评价人数、**影片/剧集类型（如喜剧、恐怖、动作等）**以及详细的剧情简介。

---

## 🌟 核心特色

1. **🚀 默认极速首选 NeoDB (源: NeoDB)**
   * 默认采用联邦宇宙开源影视数据库 **NeoDB (neodb.social)** 接口。
   * **免人机验证码**，国内直连响应速度极快（毫秒级），影视条目覆盖率极庞大（包括国内外极其小众或最新的影视）。

2. **🏷️ 影视/剧集类型（Genre）清晰展示**
   * 自动解析并展示影片或剧集的分类标签（例如 `喜剧 / 恐怖 / 动作`）。
   * 支持多语言标签智能映射，无论是电影还是电视剧都能一目了然了解类型属性。

3. **🛡️ 免验证码双源强力联合兜底机制**
   * 首次打开页面时，脚本会自动按顺序启动双源兜底：**NeoDB API $\rightarrow$ WMDB API**。
   * 全程免验证码干扰，无论哪个源出现问题，脚本都会在后台静默尝试下一个源，直至成功把数据呈现在您面前。

4. **⇄ 智能 [手动控源] 与 [一键换源]**
   * 页面渲染后，在右上角常驻 **`换源 ⇄`** 按钮。
   * 如果当前源的数据有缺失（例如电影在当前平台暂无评分或简介），您可以点击 `换源 ⇄`。此时脚本将进入**手动控源模式**，直接精准请求下一个指定的源（不再自动静默降级），以便您自主获取最想要的结果。

5. **🔗 永久常驻的直达详情链接**
   * 不论加载成功、加载中还是加载失败，右上角都永久保留直达详情页的按钮（如 **`直达NeoDB ↗`** 或 **`直达WMDB ↗`**），并且会根据您当前所选的源动态变化。

6. **⚡ 智能缓存与展开收起**
   * **缓存机制**：成功获取的数据会在本地缓存 7 天；如果是“暂无简介”等空数据，仅缓存 1 天以方便快速同步最新的数据库。
   * **折叠体验**：简介过长时会自动进行限宽折叠，并提供 `[展开]/[收起]` 按钮，不破坏原有页面布局。

## 📥 安装步骤

在浏览器中安装 [Tampermonkey (篡改猴)](https://www.tampermonkey.net/) 插件后，您可以选择以下任意一种方式安装脚本：

### 🚀 方式一：推荐（一键安装与自动更新）
直接访问 Greasy Fork 脚本主页，点击安装即可：
👉 **[Greasy Fork: 爱看机器人-影视简介](https://greasyfork.org/zh-CN/scripts/586853-%E7%88%B1%E7%9C%8B%E6%9C%BA%E5%99%A8%E4%BA%BA-%E5%BD%B1%E8%A7%86%E7%AE%80%E4%BB%8B)**

### 🛠️ 方式二：手动复制代码（本地备用）
1. 打开最新的脚本代码文件 [ikanbot-more-info.user.js](ikanbot-more-info.user.js)。
2. 复制里面的全部代码。
3. 点击浏览器右上角 Tampermonkey 图标，选择 **“添加新脚本”**。
4. 清空编辑器中的默认内容，粘贴刚才复制的全部代码并保存。

## 🛠️ 技术路线

* **NeoDB API**：无限制的开放 API，接口为干净的 JSON 元数据，不仅提供原始评分、类型和简介，响应迅速且无防爬拦截。
* **WMDB API**：第三方影视数据公益接口，提供免授权的影视条目与类型查询（限制同 IP 每 10 秒请求 1 次）。
